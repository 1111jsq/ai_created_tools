# Change: Add PPT Chart Generator

## Why
The user needs a tool to automatically generate PowerPoint presentations (`.pptx`) with charts from natural language descriptions. This allows for quick visualization of data without manual PPT editing.

## What Changes
- Introduce a new Python-based tool `PPT/`.
- Add `ppt-generator` capability.
- Implement LLM integration for data extraction.
- Implement `python-pptx` integration for chart generation.

## Impact
- Affected specs: `ppt-generator` (new).
- Affected code: New directory `PPT/` and its contents.
